# Example Module Documentation

This is the main documentation for the Example Module, demonstrating how to create comprehensive module documentation.

## Overview

The Example Module provides custom CSS effects and serves as a template for contributors.

## Features

- Custom flip animation with enhanced easing
- Styled example-box model
- Complete documentation structure
- Performance optimizations

## Quick Reference

```html
<div class="manyang-example-box manyang-h-example-flip">
    Hover me for a custom flip!
</div>
```

## Documentation Structure

This module includes:
- [effects.md](effects.md) - Technical effect specifications
- [usage.md](usage.md) - Usage examples and integration guide
