
# manyang-css-hover

[![Demo](https://img.shields.io/badge/Demo-Live-blue?style=flat-square)](./demo/)


## Overview
manyang-css-hover is a lightweight CSS library that provides a collection of hover effects to enhance user interfaces. This library is designed to be easy to use and integrate into any web project.

## Installation

You can install the package via npm:

```
npm install manyang-css-hover
```

Alternatively, you can include the CSS files directly in your project:

```html
<link rel="stylesheet" href="path/to/manyang-hover.css">
```

or for SCSS:

```scss
@import 'path/to/manyang-hover.scss';
```

## Usage

To use the hover effects, simply add the appropriate classes to your HTML elements. Here are some examples:

```html
<button class="hover-effect">Hover me!</button>
```
